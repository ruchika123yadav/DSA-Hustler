# Leaders in the array
-------------------------------------------------------------------------------------------------------------------------
## Optimal Solution
1) create a varaible for storing the max value or the leader from the right side.
2) Starting iterating the arry from right to left.
3) If the current value is greater than the max value , then add that to the list.
4) else dont add that value to that list.
5) once the loop is completed, reverse the array and return the array.


* Time Complexity : O(n)
* Space Complexity : O(1)
